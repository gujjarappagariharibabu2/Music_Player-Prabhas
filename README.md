# Simple CSS, HTML, and JavaScript Project

This project demonstrates the fundamentals of web development using CSS, HTML, and JavaScript. It includes a basic HTML structure, CSS styling for the page's layout and appearance, and JavaScript for adding interactivity and dynamic behavior.
## Project Structure

The project is structured with the following files:


index.html // Main HTML page
style.css // CSS styles for the page
script.js // JavaScript code for adding interactivity


## HTML Structure

The index.html file defines the basic structure of the web page, including the main elements like <header>, <main>, and <footer>. It also includes links to the CSS and JavaScript files.



## CSS Styling

The style.css file defines the styles for the page's layout and appearance. It includes styles for fonts, colors, margins, and other CSS properties.

css
body {
  font-family: sans-serif;
  margin: 0;
  padding: 0;
}

header {
  background-color: #f0f0f0;
  padding: 20px;
}

h1 {
  text-align: center;
  font-size: 24px;
}

p {
  margin: 20px;
  line-height: 1.5;
}

button {
  padding: 10px 20px;
  background-color: #007bff;
  color: white;
  border: none;
  cursor: pointer;
}

footer {
  background-color: #f0f0f0;
  padding: 20px;
  text-align: center;
}


## JavaScript Interactivity

The script.js file adds interactivity to the page using JavaScript. It handles the click event on the button and updates the page accordingly.

javascript
const button = document.getElementById('button');
const output = document.getElementById('output');


button.addEventListener('click', () => {
  output.textContent = 'Button clicked!';
});


This simple project demonstrates the basic usage of CSS, HTML, and JavaScript for web development. It provides a starting point for learning and exploring these fundamental technologies.
